# Trabowl Tattoo Studio Website

Premium multi-page Next.js website scaffold for a Kenya-based tattoo and piercing brand.

## Run
1. `npm install`
2. Copy `.env.example` to `.env.local` and set Supabase keys.
3. `npm run dev`

## Editable Admin Placeholders
- Phone / WhatsApp / address / email / hours: `lib/config.ts`
- Featured tattoos, testimonials, styles: `lib/data.ts`
- Artist bios, pricing, certifications: page placeholders ready for CMS wiring.

## Notes
- Place your hero loop video at `public/media/tattoo-hero.mp4`.
- Replace placeholder PDF at `public/docs/aftercare-guide-placeholder.pdf`.
- Architecture is CMS-ready and can be upgraded to Supabase tables for dynamic content.
